# Assignment Portal

A comprehensive MERN stack application for managing and evaluating student assignments.

## Tech Stack
- **Frontend**: React.js, Bootstrap
- **Backend**: Node.js, Express
- **Database**: MongoDB Atlas
- **File Storage**: Cloudinary
- **Authentication**: JWT (Admin) & Google OAuth (Faculty/Student)

## Project Structure
- `client/`: React frontend application.
- `server/`: Node.js backend API.
